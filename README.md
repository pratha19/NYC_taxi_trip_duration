This project involves predicting the yellow taxi trip duration between any two locations X and Y within NYC. 

The project is divided into these major parts:
1. Raw Data import
2. External data imports 
3. Data cleaning
4. EDA
5. Data cleaning and wranging based on EDA observations
6. Preparing data for modeling
7. Building and tuning ML models
8. Comparing the different models
9. Conclusion and futrue work
